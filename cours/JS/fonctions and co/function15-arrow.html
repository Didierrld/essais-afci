Il existe un moyen simple d'ecrire les fonctions expression.
Ce sont les fonction arrow (flêche) : 

                let mafonction = (param1, para2, ... paramN) => expression
Ceci crée une fonction qui prend les parametres de param1 à paramN, évalue l'expression et la retourne.

C'est une façon courte d'ecrire : 
                let mafonction = function (param1, para2, ... paramN)  {
                    return expression;
                };

Exemple : 

                la version arrow est :

                let prod = (a, b) => a * b;

                la version expression est :

                let prod = function(a, b) {
                    return a * b;
                };

                alert( prod(3, 6) );
Si il y a un seul parametre on peut omettre les parenthèses: 
                let mafonction = param1 => expression
Si il n y a pas de parametre, les parenthèses sont obligatoires: 
                let mafonction = () => expression

                exemple :

                let bonjour = () => alert("Bonjour mon ami")
Si le corps de la fonction est long i.e. plusieurs lignes, il faut entourer expression par des accolades et ajouter un return. 
                let mafonction = (param1,.. paramN) => {
                    // code à exécuter
                    return expression_result;
                }
